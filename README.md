# nanodip_dependencies
### Dependencies for NaniDiP

`cd /applications`

`git clone https://github.com/neuropathbasel/nanodip_dependencies.git`

`cd nanodip_dependencies`

`./all_build_x86_64.sh`

The final terminal output should read: 

`13 overlap CpGs found; done!`
